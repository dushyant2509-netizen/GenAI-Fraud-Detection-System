## Project Overview
Developed a real-time fraud detection framework using Python, Unsupervised Machine Learning, and Generative AI to identify suspicious financial transactions.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Plotly
- OpenAI/Gemini API
- Jupyter Notebook

## Key Features
- Data Cleaning and ETL
- Anomaly Detection using Unsupervised ML
- GenAI-powered transaction explanations
- Interactive Dashboard
- Workflow Automation

## Outcome
Built an intelligent fraud monitoring system capable of detecting unusual transaction patterns and providing contextual explanations for flagged transactions.
